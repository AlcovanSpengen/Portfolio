About me

Hi, I'm Alco van Spengen and I'm a junior web developer. I am learning HTML, CSS, PHP, JavaScript and much more! I have a lot of experience providing front-end development, producing high quality responsive websites and exceptional user experience. The work I provide is of highest quality, fully responsive, and tested in a wide range of devices. I take great care to ensure each project is well-documented and easily maintainable so you can enhance a website as your company grows.

I am following a coding bootcamp at Code Gorilla in Groningen. This bootcamp will take me on a journey in the world of coding and programming for about 3 months. Here I will learn how to code properly and efficiently.

My goal is to fully master HTML, CSS, PHP & JavaScript! These are the core coding languages to learn for anyone who aspires to be a web developer.
